# Nx-Pedia

Nx-Pedia is a simple and informative web-based encyclopedia designed to present structured knowledge in a clean and accessible way. It is built as a static site and hosted using GitHub Pages.

Live Site: https://nx75.github.io/Nx-Pedia/
