Riemann Sums

A Riemann sum is a method for approximating the total area underneath a curve on a graph, otherwise known as an integral. It was introduced by the German mathematician Bernhard Riemann in 1854 as a rigorous formulation of the integral concept. The idea is to partition the interval of integration into smaller subintervals, create rectangles whose heights are determined by the function at specific points within each subinterval, and then sum the areas of these rectangles.

∫ab f(x) dx = limn→∞ ∑i=1n f(xi*) Δx

For a partition of [a,b] into n subintervals of equal width Δx = (b-a)/n, with sample points xi* in each subinterval [xi-1, xi], the Riemann sum is:

S = ∑i=1n f(xi*) Δx

Types of Riemann sums:

  • Left Riemann sum: Uses left endpoints (xi-1) as sample points
  • Right Riemann sum: Uses right endpoints (xi) as sample points
  • Midpoint Riemann sum: Uses midpoints of each subinterval
  • Upper Riemann sum: Uses supremum (maximum) of f on each subinterval
  • Lower Riemann sum: Uses infimum (minimum) of f on each subinterval
Example: Approximate ∫02 x² dx using right Riemann sum with n=4.
Δx = (2-0)/4 = 0.5
Right endpoints: 0.5, 1.0, 1.5, 2.0
S = 0.5[f(0.5) + f(1.0) + f(1.5) + f(2.0)]
S = 0.5[0.25 + 1 + 2.25 + 4] = 0.5(7.5) = 3.75
Actual value: 8/3 ≈ 2.667, so this overestimates because x² is increasing.

Fundamental Theorem of Calculus

The Fundamental Theorem of Calculus (FTC) establishes the connection between differentiation and integration, two of the main concepts in calculus. It has two parts, often referred to as the First Fundamental Theorem and the Second Fundamental Theorem.

Part 1 (FTC1): If f is continuous on [a,b] and F is defined by:

F(x) = ∫ax f(t) dt

then F is continuous on [a,b], differentiable on (a,b), and F'(x) = f(x) for all x in (a,b). In other words, the function F is an antiderivative of f.

Part 2 (FTC2): If f is continuous on [a,b] and F is any antiderivative of f (i.e., F' = f), then:

∫ab f(x) dx = F(b) − F(a)

The theorem was independently developed by Isaac Newton and Gottfried Wilhelm Leibniz in the late 17th century. It transforms the problem of computing areas into finding antiderivatives, making integration computationally feasible.

Example: Evaluate ∫13 (2x + 1) dx using FTC2.
Find antiderivative: F(x) = x² + x + C
∫13 (2x + 1) dx = F(3) − F(1) = (9 + 3) − (1 + 1) = 12 − 2 = 10

Net Area vs. Total Area

The definite integral ∫ab f(x) dx gives the net area between the curve and the x-axis — area above the axis is counted as positive, area below as negative. This is different from total area, which counts all area as positive.

Net Area = ∫ab f(x) dx
Total Area = ∫ab |f(x)| dx
Example: For f(x) = sin x on [0, 2π]:
Net area = ∫02π sin x dx = 0 (positive and negative areas cancel)
Total area = ∫02π |sin x| dx = 4 (area of two humps: 2 + 2 = 4)

To find total area when a function crosses the x-axis:

  1. Find all x-intercepts (zeros) of f(x) in [a,b]
  2. Split the integral at these points
  3. Integrate |f(x)| = f(x) where f(x) ≥ 0, and −f(x) where f(x) ≤ 0
  4. Sum the absolute values

Average Value of a Function

The average value of a function f(x) over an interval [a,b] is defined as the height of a rectangle with base [a,b] that has the same area as the region under the curve. This extends the concept of averaging discrete values to continuous functions.

favg = 1⁄(b-a) ∫ab f(x) dx

The Mean Value Theorem for Integrals states that if f is continuous on [a,b], then there exists at least one number c in [a,b] such that:

f(c) = favg = 1⁄(b-a) ∫ab f(x) dx

Geometrically, this means there is a rectangle with base [a,b] and height f(c) that has exactly the same area as the region under the curve.

Example: Find the average value of f(x) = x² on [0,2] and find c where f(c) equals this average.
favg = 1/(2-0) ∫02 x² dx = (1/2)[x³/3]02 = (1/2)(8/3) = 4/3 ≈ 1.333
Find c: f(c) = c² = 4/3 → c = √(4/3) = 2/√3 ≈ 1.155, which lies in [0,2]

Properties of Definite Integrals

Definite integrals satisfy several important properties that simplify calculations and theoretical manipulations. These properties follow directly from the definition of the integral as a limit of Riemann sums.

PropertyFormula
Zero width∫aa f(x) dx = 0
Reverse limits∫ab f(x) dx = −∫ba f(x) dx
Constant multiple∫ab k·f(x) dx = k ∫ab f(x) dx
Sum/Difference∫ab [f(x) ± g(x)] dx = ∫ab f(x) dx ± ∫ab g(x) dx
Additivity of intervals∫ab f(x) dx + ∫bc f(x) dx = ∫ac f(x) dx
Inequality (monotonicity)If f(x) ≤ g(x) on [a,b], then ∫ab f(x) dx ≤ ∫ab g(x) dx
BoundsIf m ≤ f(x) ≤ M on [a,b], then m(b-a) ≤ ∫ab f(x) dx ≤ M(b-a)
Even functionsIf f is even, ∫-aa f(x) dx = 2∫0a f(x) dx
Odd functionsIf f is odd, ∫-aa f(x) dx = 0
Periodic functionsIf f has period T, ∫aa+T f(x) dx = ∫0T f(x) dx
Linearity in action: ∫01 (3x² + 2x) dx = 3∫01 x² dx + 2∫01 x dx = 3(1/3) + 2(1/2) = 1 + 1 = 2

Comparison Theorem

The Comparison Theorem for integrals provides a way to bound an integral by comparing the integrand to simpler functions. It's particularly useful when the exact value is difficult to compute but bounds are sufficient, and for establishing convergence of improper integrals.

Basic Comparison: If f(x) ≤ g(x) for all x in [a,b], then:

∫ab f(x) dx ≤ ∫ab g(x) dx

Absolute Value Comparison: Since −|f(x)| ≤ f(x) ≤ |f(x)|, we have:

|∫ab f(x) dx| ≤ ∫ab |f(x)| dx

Bounding Theorem: If m ≤ f(x) ≤ M on [a,b], then:

m(b−a) ≤ ∫ab f(x) dx ≤ M(b−a)

For improper integrals: If 0 ≤ f(x) ≤ g(x) on [a,∞) and ∫a∞ g(x) dx converges, then ∫a∞ f(x) dx converges. Conversely, if ∫a∞ f(x) dx diverges, then ∫a∞ g(x) dx diverges.

Example: Show that 1 ≤ ∫01 √(1+x³) dx ≤ √2.
On [0,1], x³ ranges from 0 to 1, so 1 ≤ 1+x³ ≤ 2
Taking square roots: 1 ≤ √(1+x³) ≤ √2
Integrating: 1·(1-0) ≤ ∫01 √(1+x³) dx ≤ √2·(1-0)
Therefore: 1 ≤ ∫01 √(1+x³) dx ≤ √2 ≈ 1.414
Example - Improper integrals: Determine if ∫1∞ e-x² dx converges.
For x ≥ 1, e-x² ≤ e-x (since x² ≥ x)
∫1∞ e-x dx = limb→∞ [−e-x]1b = 0 + e-1 = 1/e (converges)
By comparison, since 0 ≤ e-x² ≤ e-x and ∫ e-x converges, ∫ e-x² also converges.
