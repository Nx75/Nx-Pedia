Definition of an Antiderivative

An antiderivative, also called an indefinite integral or primitive function, is the inverse operation of differentiation. Formally, a function F is an antiderivative of f on an interval I if:

F'(x) = f(x) for all x in I

In other words, differentiating F returns the original function f. The process of finding antiderivatives is called integration. While differentiation has a unique result for a given function, antiderivatives are not unique — if F is an antiderivative of f, then F(x) + C is also an antiderivative for any constant C, since the derivative of a constant is zero.

The notation for the general antiderivative (indefinite integral) is:

∫ f(x) dx = F(x) + C

where ∫ is the integral sign (introduced by Leibniz in 1675, adapted from the long s meaning "summa" or sum), f(x) is the integrand, dx indicates integration with respect to x, and C is the constant of integration. This relationship is formalized by the Fundamental Theorem of Calculus, which connects differentiation and integration as inverse operations.

Power Rule

The power rule for antiderivatives is derived directly from the power rule for derivatives. Since d/dx [xⁿ⁺¹/(n+1)] = xⁿ for n ≠ -1, the reverse holds:

∫ xⁿ dx = (xⁿ⁺¹)/(n+1) + C,    n ≠ -1

Proof: Differentiate the right side: d/dx [(xⁿ⁺¹)/(n+1) + C] = (n+1)xⁿ/(n+1) = xⁿ, confirming it's correct. The rule fails for n = -1 because division by zero occurs. This special case gives the natural logarithm.

Examples:
∫ x³ dx = x⁴/4 + C
∫ x⁻² dx = x⁻¹/(-1) + C = -1/x + C
∫ √x dx = ∫ x¹/² dx = x³/²/(3/2) + C = (2/3)x³/² + C

Constant of Integration

The constant of integration C is essential because differentiation eliminates constants. Geometrically, the family of antiderivatives F(x) + C represents vertical translations of the same curve. All these functions have identical slope at each x because their derivatives are the same.

Why C matters: Without C, the relationship between differentiation and integration would not be one-to-one. In applications, initial conditions determine the specific value of C. For example, if velocity v(t) = 2t is known, position s(t) = ∫ 2t dt = t² + C. If we know s(0) = 5, then C = 5, giving the particular solution s(t) = t² + 5.

Reverse Chain Rule

The reverse chain rule (also called integration by inspection) identifies when an integrand is exactly the derivative of a composite function. From the chain rule: d/dx [F(g(x))] = F'(g(x)) · g'(x). Therefore:

∫ F'(g(x)) · g'(x) dx = F(g(x)) + C

In practice, we look for a function and its derivative present in the integrand.

Common patterns:
∫ [f(x)]ⁿ · f'(x) dx = [f(x)]ⁿ⁺¹/(n+1) + C (n ≠ -1)
∫ f'(x)/f(x) dx = ln|f(x)| + C
∫ e^{f(x)} · f'(x) dx = e^{f(x)} + C
Examples:
∫ (3x²) · (x³+1)⁴ dx: Let u = x³+1, du = 3x² dx → ∫ u⁴ du = u⁵/5 + C = (x³+1)⁵/5 + C
∫ (cos x)/(sin x) dx: f(x) = sin x, f'(x) = cos x → ∫ du/u = ln|sin x| + C

Substitution Basics

When the reverse chain rule isn't directly applicable, u-substitution (change of variables) systematically transforms the integral. The steps are:

  1. Choose a substitution u = g(x) where g is differentiable.
  2. Compute du = g'(x) dx and solve for dx.
  3. Rewrite the entire integral in terms of u and du.
  4. Integrate with respect to u.
  5. Substitute back to x.
∫ f(g(x)) · g'(x) dx = ∫ f(u) du, where u = g(x)
Example: ∫ 2x √(x²+1) dx
Let u = x²+1, du = 2x dx
Then ∫ √u du = ∫ u¹/² du = (2/3)u³/² + C = (2/3)(x²+1)³/² + C

For definite integrals, limits must also change: ∫_{x=a}^{x=b} f(g(x)) g'(x) dx = ∫_{u=g(a)}^{u=g(b)} f(u) du

Initial Value Problems

Initial value problems (IVPs) combine antiderivatives with given conditions to find particular solutions. These arise naturally in physics (position from velocity), economics (cost from marginal cost), and population modeling.

General approach:
1. Find the general antiderivative including +C.
2. Use the initial condition F(x₀) = y₀ to solve for C.
3. Write the particular solution.
Detailed example: A particle moves with velocity v(t) = 3t² - 2t m/s. At t = 1 s, its position s = 4 m. Find s(t).
s(t) = ∫ (3t² - 2t) dt = t³ - t² + C
Using s(1) = 4: 1³ - 1² + C = 4 → 0 + C = 4 → C = 4
Therefore s(t) = t³ - t² + 4
Another example: Given dy/dx = eˣ + 1/x and y(1) = 2e. Find y.
y = ∫ (eˣ + 1/x) dx = eˣ + ln|x| + C
y(1) = e¹ + ln 1 + C = e + 0 + C = 2e → C = e
Thus y = eˣ + ln|x| + e
The family of antiderivatives for f(x)=2x (F(x)=x²+C) differ by vertical shifts. The constant of integration determines which specific curve.
U-substitution transforms the integral into a basic form.
Initial value problems determine the specific constant.
